UFC Beginner's Guide - Site Plan

Manuel Baez

WDD 131 - Final Project

Purpose

The purpose of this website is to introduce complete beginners to the UFC by explaining the rules, weight classes, fighting styles, scoring system, and how to understand events. It will simplify MMA concepts to help new fans watch confidently.

Audience

This site is designed for teens and adults who are curious about the UFC but have little or no prior knowledge. It is written in simple, friendly language for beginners.

Dynamic JavaScript Elements

The site will include several dynamic features:

  • Fighter Style Filter: A set of buttons will filter a list of fighters by style (Striker, Wrestler, BJJ, etc.) using arrays and the filter() method.
  • Random UFC Fact Generator: When a button is clicked, a random fact object will be selected and displayed using conditional logic and DOM updates.
  • Mobile Menu Toggle: A hamburger menu will open/close using event listeners and DOM manipulation for responsive navigation.

Logo

The site's logo is a clean, modern graphic featuring UFC-style gloves and bold typography. This will appear on every page.

Color Scheme

Primary colors for the website:

  • Red: #D62828
  • Dark Gray / Black: #1A1A1A
  • Light Gray: #E6E6E6
  • White: #FFFFFF

Fonts

  • Headers: Montserrat
  • Body Text: Roboto

Content

Home Page

Welcome to the UFC Beginner Guide! This site will teach you what MMA is, how UFC events work, and what you should know before watching your first fight. Learn the basics and explore the world of mixed martial arts.

Child Page (Basics & Rules)

On this page, users can explore weight classes, fighting styles (Striking, Wrestling, BJJ), how UFC scoring works, and what a fight card is. The page includes dynamic filtering and a random fact feature.
